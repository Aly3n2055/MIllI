## 🔧 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Aly3n2077/MIllI.git
   cd MIllI
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the project:
   ```bash
   npm run build
   ```

4. Start the application:
   ```bash
   npm start
   ```

## 🧪 Running Tests

To run tests, execute:
```bash
npm test
```
